# Installation

A quick tutorial to walk through installing the packages, as well as the supported plugins.

## Installing the packages

```bash
npm install @radix-ng/primitives @angular/cdk
```

## Import the parts

```typescript
import { Component } from '@angular/core';
import { bootstrapApplication } from '@angular/platform-browser';
import 'zone.js';
import { RdxLabelDirective } from '@radix-ng/primitives/label';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [RdxLabelDirective],
  template: `
    <h1>Hello from {{ name }}!</h1>
    <a target="_blank" href="https://angular.dev/overview">Learn more about Angular</a>
    <div>
      <label rdxLabel htmlFor="uniqId">First Name</label>
      <input class="Input" id="uniqId" type="text" />
    </div>
  `
})
export class App {
  name = 'Angular';
}

bootstrapApplication(App);
```
