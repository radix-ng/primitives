# Dialog

<Description>A window overlaid on either the primary window or another dialog window, rendering the content underneath inert.</Description>

## Anatomy

Import all parts and assemble them together.

```html
<div rdxDialogRoot>
    <button rdxDialogTrigger>Open</button>

    <ng-template rdxDialogPortalPresence>
        <div rdxDialogPortal>
            <div rdxDialogBackdrop></div>
            <div rdxDialogPopup>
                <h2 rdxDialogTitle></h2>
                <p rdxDialogDescription></p>
                <button rdxDialogClose></button>
            </div>
        </div>
    </ng-template>
</div>
```

Import the directives via the `dialogImports` array (or the individual `Rdx*` classes).

```typescript
import { dialogImports } from '@radix-ng/primitives/dialog';

@Component({
    imports: [...dialogImports]
})
export class AppDialogComponent {}
```

## API Reference

### Root

Groups all parts of the dialog. Owns the open state and provides the dialog context.

<PropsTable name="RdxDialogRoot" />

### Trigger

A button that opens the dialog.

<PropsTable name="RdxDialogTrigger" />

<DataAttributesTable attributes={[
    { name: '[data-state]', value: '"open" | "closed"' }
]} />

### Portal

Moves the dialog to a different part of the DOM. By default, the portal element is appended to `document.body`.

<PropsTable name="RdxDialogPortal" />

### Backdrop

An overlay displayed beneath the dialog popup.

<DataAttributesTable attributes={[
    { name: '[data-state]', value: '"open" | "closed"' },
    { name: '[data-starting-style]', value: 'Present while the dialog is animating in.' },
    { name: '[data-ending-style]', value: 'Present while the dialog is animating out.' }
]} />

### Popup

A container for the dialog contents.

<PropsTable name="RdxDialogPopup" />

<DataAttributesTable attributes={[
    { name: '[data-state]', value: '"open" | "closed"' },
    { name: '[data-starting-style]', value: 'Present while the dialog is animating in.' },

]} />

### Close

A button that closes the dialog.

Adheres to the [Dialog WAI-ARIA design pattern](https://www.w3.org/WAI/ARIA/apg/patterns/dialog-modal).
