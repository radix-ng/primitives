# Toggle Group

<Description>A set of two-state buttons that can be toggled on or off.</Description>

<ComponentPreview name="toggle-group" file="toggle-group-demo" />

<FeatureList items={['Full keyboard navigation.',
    'Supports horizontal/vertical orientation.',
    'Support single and multiple pressed buttons.',
    'Can be controlled or uncontrolled.',]} />

## Import

Get started with importing the directives:

```typescript
import { RdxToggleGroup } from '@radix-ng/primitives/toggle-group';
import { RdxToggle } from '@radix-ng/primitives/toggle';
```

## Anatomy

```html
<div rdxToggleGroup [value]="['center']">
  <button rdxToggle value="left"></button>
  <button rdxToggle value="center"></button>
  <button rdxToggle value="right"></button>
</div>
```

## API Reference

### Root

`RdxToggleGroup`

Contains all the parts of a toggle group. The `value` is always an array.

### Item

`RdxToggle`

An item in the group — see the Toggle page. Each item needs a `value`.

## Accessibility

Uses [roving tabindex](https://www.w3.org/TR/wai-aria-practices-1.2/examples/radio/radio.html) to manage focus movement among items.
