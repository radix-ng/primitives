# Preview Card

<Description>For sighted users to preview content available behind a link.</Description>

<ComponentPreview name="preview-card" file="preview-card-demo" />

<FeatureList items={[
    'Can be controlled or uncontrolled.',
    'Customize side, alignment, offsets, collision handling.',
    'Optionally render a pointing arrow.',
    'Supports custom open and close delays.',
    'Ignored by screen readers.'
]}/>

## Import

Get started with importing the directives:

```typescript
import {
  RdxPreviewCardModule
} from '@radix-ng/primitives/preview-card';
```

## Anatomy

```html
<ng-container rdxPreviewCardRoot>
    <a rdxPreviewCardTrigger></a>

    <ng-template rdxPreviewCardPortalPresence>
        <div rdxPreviewCardPortal>
            <div rdxPreviewCardPositioner>
                <div rdxPreviewCardPopup>
                    Preview Card Content
                    <div rdxPreviewCardArrow></div>
                </div>
            </div>
        </div>
    </ng-template>
</ng-container>
```
## API Reference

### Root

`RdxPreviewCardRoot`

Contains all the parts of a preview card.

<PropsTable name="RdxPreviewCardRoot" />

### Trigger

`RdxPreviewCardTrigger`

The link or element that opens the preview card. By default, the `PreviewCardPositioner` will position itself against the trigger.

### Positioner

`RdxPreviewCardPositioner`

Positions the preview card against its active trigger.

<PropsTable name="RdxPreviewCardPositioner" />

### Popup

`RdxPreviewCardPopup`

The popup element rendered when the preview card is open.

<PropsTable name="RdxPreviewCardPopup" />

### Arrow

`RdxPreviewCardArrow`

An optional arrow element to render alongside the preview card. This can be used to help visually link the trigger with the popup.

<PropsTable name="RdxPreviewCardArrow" />
